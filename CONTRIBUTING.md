# Contributing to OpenSCAD Embroidery Hoop Generator

Thank you for your interest in contributing to this project! This is a derivative work building on the excellent original design by Braden Sunwold.

## How to Contribute

### Reporting Issues
- Check existing issues before creating a new one
- Provide clear description of the problem
- Include OpenSCAD version and parameter settings
- Screenshots or STL files help immensely

### Suggesting Features
- Open an issue with the "enhancement" label
- Describe the use case and benefit
- Consider backward compatibility

### Code Contributions

#### Prerequisites
- OpenSCAD 2021.01 or newer
- Basic understanding of OpenSCAD syntax
- Git for version control

#### Process
1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Make your changes
4. Test thoroughly with various parameter combinations
5. Update documentation if needed
6. Commit with clear messages (`git commit -m 'Add amazing feature'`)
7. Push to your fork (`git push origin feature/amazing-feature`)
8. Open a Pull Request

#### Code Style
- Use descriptive variable names
- Add comments for complex calculations
- Group related parameters in Customizer
- Include parameter validation where appropriate
- Echo important warnings to console

#### Testing Checklist
- [ ] Parameters work across full range
- [ ] No geometry errors in OpenSCAD
- [ ] STL exports cleanly
- [ ] Printability verified (if possible)
- [ ] Documentation updated
- [ ] Console output appropriate

## Priority Areas for Contribution

### High Priority
- Hardware size dropdown presets (M3-M6, imperial sizes)
- Additional groove patterns/styles
- Alternative clamping mechanisms
- Improved parameter validation
- Bug fixes

### Medium Priority
- Print time/material estimates
- Pre-configured hoop size templates
- Decorative pattern options
- Multi-groove configurations
- Assembly instructions/diagrams

### Nice to Have
- Magnetic closure variations
- Snap-fit alternatives
- Integrated fabric grippers
- Stand/display base options

## Questions?

Open an issue with the "question" label or start a discussion.

## Code of Conduct

Be respectful, constructive, and collaborative. We're all here to make awesome embroidery hoops!
